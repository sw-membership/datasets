# datasets

주식 긍부정 분석을 위한 데이터셋입니다.

## labels

레이블은 다음으로 처리되었습니다.

- -1: 부정
- 0: 중립
- 1: 긍정

## example

```shell
# data.csv
text,labels
hello world,-1
hi there,0
```
